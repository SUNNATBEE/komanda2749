### Что сверстать

- **Hero** — заголовок «Авто в кредит для такси», форма заявки (имя, телефон)
- **Блок условий** — первый взнос, ежемесячный платёж, срок (карточки)
- **Авто в наличии** — список моделей для такси (карточка + цена + «Оставить заявку»)
- **Банки-партнёры** — логотипы
- **Шаги оформления** — «Заявка → Одобрение → Получение авто» (3-4 шага)
- **FAQ** — частые вопросы (аккордеон)

### Правила

- Используйте **Tailwind CSS**
- Переиспользуйте карточки/стили с главной страницы (единый вид)
- Проверьте через `npm run dev` — переходы по меню должны работать

## GitHub — ваша ветка: `feature/mansur`

> Полная шпаргалка: [`docs/GIT_GUIDE.ru.md`](../../GIT_GUIDE.ru.md)

### Ежедневный цикл

```bash
# 1. Обновить main и перейти в свою ветку
git checkout main && git pull origin main
git checkout feature/mansur
git merge origin/main

# 2. После правок в TaxiCreditPage.jsx
git add src/pages/Mansur/
git commit -m "feat(mansur): страница такси в кредит"
git push -u origin feature/mansur

# 3. На GitHub → Pull Request → feature/mansur → main → Merge
```

| Команда | Действие |
|---------|----------|
| `git pull` | Скачать новые изменения с GitHub |
| `git add .` | Подготовить файлы к сохранению |
| `git commit -m "..."` | Сохранить изменения локально |
| `git push` | Отправить коммиты на GitHub |
| **Merge (PR)** | Объединить ветку с `main` после проверки |
